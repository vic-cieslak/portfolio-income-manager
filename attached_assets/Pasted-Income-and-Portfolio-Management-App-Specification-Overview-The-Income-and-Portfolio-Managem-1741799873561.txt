Income and Portfolio Management App - Specification

## Overview

The **Income and Portfolio Management App** is a web application designed to help track daily income and manage financial portfolios, including cryptocurrency and traditional cash/bank accounts. The application will provide a calendar-based income visualization and real-time asset tracking.

## Technology Stack

- **Backend:** Django (Python)
- **Frontend:** Django templates (or Vue.js if necessary for interactivity)
- **Database:** PostgreSQL (or SQLite for development)
- **Authentication:** No user registration (single-user system)
- **Third-Party APIs:** Cryptocurrency price tracking API (e.g., CoinGecko, Binance API)
- **Deployment:** Docker, Nginx, Gunicorn

## Features

### 1. Authentication (Single-User System)

- No user registration or multi-user support needed
- Simple admin login to access the dashboard

### 2. Income Tracking

- Ability to **add** daily income entries with:
    - Amount
    - Date
    - Income category (salary, freelance, investment, etc.)
    - Description (optional)
- Users can **view income** in a **calendar view**:
    - Each day shows total income earned
    - Option to filter by month/year
- Ability to **edit** and **delete** income entries

### 3. Portfolio Management

#### Cryptocurrency Tracking

- Ability to **add** cryptocurrency assets:
    - Coin name/symbol
    - Quantity owned
    - Acquisition cost (optional)
- The system will **fetch real-time prices** via API
- Ability to **view portfolio value** updated in real-time
- Option to **track historical value**

#### Cash & Bank Account Tracking

- Ability to **add** bank/cash accounts:
    - Account name (e.g., "Savings Account", "Cash Wallet")
    - Current balance
- Ability to **update balances** manually
- Overview of **total net worth** (crypto + bank + cash)

### 4. Dashboard & Reporting

- A dashboard summarizing:
    - Monthly income
    - Total portfolio value (crypto + cash/bank)
    - Recent income entries
- Basic charts:
    - Monthly income trend
    - Portfolio allocation breakdown

### 5. API Integrations

- **Cryptocurrency Price API** to fetch real-time coin values

### 6. Additional Considerations

- **Dark mode first UI**

## Development Roadmap

### Phase 1: MVP (Minimal Viable Product)

- Admin authentication (single-user system)
- Income tracking with calendar view
- Manual bank/cash tracking
- Crypto portfolio tracking (real-time updates)
- Basic dashboard with income and portfolio summary
